# Figuras opcionais do roteiro

O arquivo `../roteiro-4-aulas-api-mobile.md` funciona sozinho com **diagramas Mermaid** (visiveis no GitHub).

Se quiser **imagens PNG** para Google Slides ou PowerPoint, voce pode:

1. Exportar os diagramas Mermaid a partir de editores online (Mermaid Live Editor), ou
2. Criar slides com o mesmo conteudo dos quadros do roteiro.

Sugestao de nomes (se adicionar arquivos aqui):

- `roteiro-cronograma-4-aulas.png` — visao geral das 4 aulas
- `roteiro-fluxo-mvp.png` — fluxo Usuario, App, API, MySQL
